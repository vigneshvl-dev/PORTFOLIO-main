# Brick Breaker Game 🎮

A classic arcade Brick Breaker game built with HTML5 Canvas and vanilla JavaScript. Features modern design, smooth physics, and responsive controls.

## 🎯 Features

- **Classic Gameplay**: Break all the bricks without losing the ball
- **Smooth Physics**: Realistic ball movement and collision detection
- **Level Progression**: Multiple levels with increasing difficulty
- **Lives System**: 3 lives to complete each level
- **Responsive Controls**: Keyboard (Arrow keys + Space) and Mouse support
- **Modern UI**: Glassmorphism effects, gradients, and neon glows
- **Score Tracking**: Real-time score, lives, and level display

## 🎮 How to Play

1. **Move the Paddle**: Use Arrow Keys (← →) or move your mouse
2. **Launch the Ball**: Press SPACE or Click
3. **Break the Bricks**: Hit all bricks to advance to the next level
4. **Don't Miss**: Keep the ball from falling off the bottom
